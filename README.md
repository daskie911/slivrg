# 🤖 Telegram Subscription Bot

Бот для управления платными подписками на приватный Telegram-канал с оплатой через **Telegram Stars**.

## ✨ Основные возможности

### Для пользователей:
- 💳 **Оплата через Telegram Stars** — встроенная платёжная система Telegram
- 🔗 **Персональные invite-ссылки** — одноразовые, действуют 30 минут
- 📅 **Автоматическое продление** — уведомления за 3 дня до окончания
- 🔐 **Защита от передачи** — ссылка работает только для покупателя
- ⏰ **Гибкие сроки** — по умолчанию 30 дней, настраивается администратором

### Для администраторов:
- 🎛️ **Админ-панель** с полным управлением
- 📊 **Статистика** — активные подписчики, доход, истекающие подписки
- 💰 **Управление ценой** — изменение стоимости подписки в реальном времени
- 📅 **Настройка сроков** — изменение длительности подписки
- 🔍 **Поиск пользователей** — просмотр информации о любом подписчике
- 📢 **Рассылка** — отправка сообщений всем активным подписчикам
- 🗑️ **Управление доступом** — удаление подписок и кик из канала
- 💵 **Учёт дохода** — автоматическое логирование всех платежей
